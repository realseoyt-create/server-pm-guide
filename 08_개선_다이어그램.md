```mermaid

    LLM --> P3["중복 실행 방지\n누락 탐지\n(flock 없음)"]

    LLM --> P5["오류 알림\n미설정 탐지"]

    P1 & P2 & P3 & P4 & P5 --> OUT["개선 가이드 출력\n· 항목별 문제 설명\n· 수정 예시 제공\n· 우선순위 분류"]
```

## 개선 효과 요약

| | AS-IS | TO-BE |
|--|-------|-------|
| **비교 시점** | Failover 이후 | Failover 이전 |
| **문제 발견** | 장애 발생 후 | 사전 인지 |
| **Crontab 품질** | 확인 안 함 | 🤖 LLM이 자동 점검 |
| **대응 방식** | 긴급 수동 조치 | 선제 계획 수립 |
| **담당자 확인** | Failover 중 (긴박) | Failover 전 (여유) |
| **고아 프로세스** | Failover 후 발견 | 사전 탐지·조치 |
