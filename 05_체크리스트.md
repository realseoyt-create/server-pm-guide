## ⚠️ 특별 주의 항목 (반드시 이중 확인)

| # | 항목 | 이유 |
|---|------|------|
| 1 | Oracle Listener 포트 **1621** | 표준 1521이 아님. 모든 포트 확인 시 1621 사용 |
| 2 | **PJOSCD** 서비스 Listener 등록 여부 | 미등록 시 앱/RVD에서 DB 접속 불가 |
| 3 | **Cron 재활성화** 후 내용 백업본과 대조 | 복구 시 가장 많이 빠뜨리는 항목 |
| 4 | DB 동기화 **재개 확인** | 중단만 하고 재개를 잊는 경우 있음 |
| 5 | RVD는 Oracle **OPEN 이후** 기동 | Oracle 미기동 상태에서 RVD 올리면 접속 오류 |
